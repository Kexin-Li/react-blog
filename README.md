# react-blog

A React &amp; Redux Blog Demo.

The backend API is from here: https://reduxblog.herokuapp.com/ ‚ù§

## Usage

clone the repo:

```
git clone https://github.com/Kexin-Li/react-blog.git
```

install the dependence:

```
npm install
```

have fun:

```
npm start
```

see http://localhost:8080 for more details.
